Example MaterialX Files

The MaterialX distribution includes libraries of example materials based on the Standard Surface, UsdPreviewSurface, and glTF PBR shading models, as well as an extensive Test Suite of examples that leverage each feature in the MaterialX standard.

Additionally, AMD maintains the GPUOpen MaterialX Library, providing a wide variety of MaterialX examples under a Creative Commons license.

Standard Surface Examples

The Materials/Examples/StandardSurface folder of the MaterialX distribution provides examples of uniform, procedural, and textured materials based on the Standard Surface shading model, which was originally developed by Autodesk. See the librares/bxdf folder for the graph definition of this shading model and others.

UsdPreviewSurface Examples

The Materials/Examples/UsdPreviewSurface folder of the MaterialX distribution provides examples of uniform and textured materials based on the UsdPreviewSurface shading model, which was originally developed by Pixar. See the librares/bxdf folder for the graph definition of this shading model and others.

glTF PBR Examples

The Materials/Examples/GltfPbr folder of the MaterialX distribution provides a simple material example based on the glTF PBR shading model, which was originally developed by Khronos. See the librares/bxdf folder for the graph definition of this shading model and others.

Test Suite Examples

The Materials/TestSuite folder of the MaterialX distribution provides example materials that leverage each node and feature in the MaterialX standard.

AMD GPUOpen MaterialX Library

AMD maintains the GPUOpen MaterialX Library, providing a wide variety of MaterialX examples under a Creative Commons license. The library is well-organized and searchable, with support for real-time web preview and contributions from the community.
